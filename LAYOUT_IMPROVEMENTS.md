# 🎨 Улучшения компоновки игры

## ✅ Выполненные изменения

### 1. Сокращено количество раундов
**Было:** 16 раундов  
**Стало:** 5 раундов  

**Причина:** Упрощение отладки и тестирования игры.

**Изменение:**
```typescript
// client/src/pages/Play.tsx
const [totalRounds] = useState(5)  // было 16
```

---

### 2. Уменьшены отступы между элементами

Теперь **не нужно скроллить** после просмотра фотографии - все элементы управления находятся на экране!

#### Изменения в отступах:

**`.play-content` (основной контейнер):**
```css
/* Было */
padding: 70px 320px;
gap: 20px;

/* Стало */
padding: 30px 320px;  /* уменьшен верхний отступ с 70px до 30px */
gap: 10px;            /* уменьшен gap между элементами с 20px до 10px */
```

**`.play-guess-section` (секция "Где это?"):**
```css
/* Было */
margin-top: 200px;  /* огромный отступ! */

/* Стало */
margin-top: 40px;   /* компактно */
```

**`.play-selection-area` (область выбора этажа):**
```css
/* Было */
margin-top: 69px;

/* Стало */
margin-top: 30px;
```

**`.play-question` (заголовок "Где это?"):**
```css
/* Было */
margin: 0 0 20px 0;

/* Стало */
margin: 0;  /* убран нижний отступ */
```

## 📱 Преимущества

1. **Быстрая отладка:** 5 раундов вместо 16 - тестирование в 3 раза быстрее
2. **Компактность:** Все на экране, не нужно скроллить
3. **Удобство:** Сразу видно фото и элементы управления
4. **Лучший UX:** Плавный игровой процесс без отвлечений
5. **Больше пространства:** Для контента вместо пустых отступов
